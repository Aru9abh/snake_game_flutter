# 🐍 Snake Game using Flutter

This is a classic Snake Game built using **Flutter** and **Dart**.

## 🎯 Features
- Simple UI with directional controls
- Automatic food generation
- Game Over detection
- Fully responsive layout

git clone https://github.com/Aru9abh
